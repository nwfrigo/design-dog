# Brand Assets

Place your brand assets in this folder:

## Logos
- `logo-black.svg` - Black version of the logo
- `logo-orange.svg` - Orange/primary color version of the logo
- `logo-white.svg` - White version (for dark backgrounds)

## Usage
These assets are served from `/assets/brand/` in the app.

Example: `/assets/brand/logo-black.svg`
